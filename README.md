# README Generator

Generate professional README files for your GitHub projects in seconds.

**🌐 Live site:** https://kik369.github.io/readme-generator/

## Features

- ✅ Real-time README preview
- ✅ Custom badges selector (Python, MIT License, CI/CD, Version, Downloads)
- ✅ Features list builder
- ✅ Installation and Quick Start sections
- ✅ Copy to clipboard / Download as .md file
- ✅ Built-in donation support (Ko-fi)
- ✅ Dark theme (matches GitHub's style)

## Tech

- Pure HTML/CSS/JavaScript — zero dependencies
- Single file (index.html) — deploy anywhere
- Fully client-side — no server needed
